# Healthcare Diagnosis Prediction System

A machine learning system for predicting disease diagnosis based on patient data, with an interactive Streamlit web interface.

## Features
- Predictive model using Random Forest Classifier
- Interactive web interface with Streamlit
- Model interpretability and feature importance
- Health metrics visualization
- Educational recommendations
- Synthetic data generation for demonstration
